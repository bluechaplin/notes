# 如何在typora中使用表情

## 一、前言

使用markdown记录笔记的时候，总觉得自己的笔记太单调了，但是看别人的笔记总觉得很好看，不妨在熟练的情况下，在自己的笔记中多插入一些小表情，让自己的笔记看起来更加的有趣

## 二、使用

### 2.1、菜单栏

直接使用`菜单栏`-->`编辑`-->`表情与符号`来添加自己想要的符号和表情

### 2.2、markdown语法

使用markdown的语法来添加表情

- People

  | 表情                    | 代码                      |
  | ----------------------- | ------------------------- |
  | :smile:             	| `:smaile:`                |
  | :disappointed_relieved: | `:disappointed_relieved:` |
  | :smirk:                 | `:smirk:`                 |
